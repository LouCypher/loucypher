
How to use this button:

1. Right click on any buttons made with CB or CB2 (untested)
2. Select 'Copy to clipboard'

or

1. Right click on any custom buttons links made for CB or CB2
   (will take times for popup menu to show up if the code is too long)
2. Select 'Copy Link Location'

and then

3. Click this button again to save it as an XML file
